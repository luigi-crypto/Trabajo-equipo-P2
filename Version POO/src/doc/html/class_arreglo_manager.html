Detailed Description

Clase encargada de la gestión, generación y visualización de datos.

Esta clase encapsula un vector dinámico de enteros y proporciona herramientas para su creación mediante menús interactivos, asegurando la integridad de los datos.

◆ generarInteractivo()

void ArregloManager::generarInteractivo ( )

Genera el arreglo usando el menú interactivo original.

Menú interactivo para configurar y generar el arreglo de datos.

Permite definir el tamaño y si se aceptan duplicados o no.

El método permite seleccionar tres tipos de dimensiones:

  1. $N$ lineal.
  2. $N \times N$ (cuadrático).
  3. $N \times M$ (rectangular).
  • Implementa dos modos de generación:
  • Repetidos: Usa std::uniform_int_distribution para llenar el vector.
  • Sin Repetidos: Crea un "pool" de valores, aplica un std::shuffle y hace un resize al tamaño deseado.
Note
Límite de seguridad para evitar desbordamientos de memoria.

void ArregloManager::mostrar ( ) const

Muestra el contenido del arreglo en consola.

Imprime los elementos del arreglo en la salida estándar.

Incluye una restricción visual para no saturar la pantalla con arreglos gigantes.

Por motivos de rendimiento y legibilidad, si el arreglo es mayor a 50 elementos, solo se muestran los primeros 50 y se indica que el resto ha sido truncado.
